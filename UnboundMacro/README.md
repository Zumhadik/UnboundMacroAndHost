# unboundmacro

tiny windows app for recording and replaying mouse/keyboard macros.

make a .macros file, hit the record hotkey, do your stuff, hit it again to stop. then play it back. loop toggle is in the ui if you want it to repeat.

# default binds

- f8 start/stop recording
- f10 start/stop playback

you can rebind both from the buttons in the window. escape cancels rebinding.

# build

open UnboundMacro.sln in visual studio and build (x64).  

output ends up in bin\Release / bin\Debug
